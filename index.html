World Wide Web (www) and HTML (Hypertext Markup Language)

World Wide Web uses HTML to connect the webpages. HTML is the heart of the web, its text content, markup (how it looks), references to documents (images and videos), and links to other web pages.

Tags

Tags are used to create the HTML code. Depending on the tag, you will have opening and/or closing tags, some tags we have used so far in module 0 are bold, emphasis, anchor, and linkable tags consisting both of websites or images.

Structure and Style

Languages

There are 3 types of Code Language, HTML (Hypertext Markup Language) ,CSS (Cascading Style Sheet), and JS (Java Script). HTML controls the structure of the page while CSS controls the style of the page.

Structure

The DOM (Document Object Model) is what they call the tree like structure of the page. Both HTML and CSS both are languages that based on syntax and rules. It contains opening tag, content and closing tag. Each tag has a name (type) such as paragraph, image, or division. Content can be empty text or other element. Documents written in HTML defines the content and structure.

Style

CSS is the language of style in coding. With CSS, you can define the font, color, border, position, etc. You can boxify a design, usually separated by header, body and footer.

Reflection and Practice

Types of tools used for Coding

There are 3 type of tools used for coding. Scratch Pad, Code Pen, and Sublime. On speed of feedback, Scratchpad is immediate, Code Pen is 2-3 seconds, and Sublime is on page refresh. On ease of sharing Scratchpad is impossible, Code Pen is easy but requires an account, Sublime is difficult. On power and professionalism, Scratch pad is low, Code Pen is very high, and Sublime is the highest. And on the number of languages, Scratchpad has 2 (HTML and CSS), Code pen has 3 (HTML, CSS, and JS), while Sublime does all the languages.

CSS Style

CSS is language to add style to your HTML. One key feature of CSS is inheritance. This the mechanism by which properties are applied to not only to the specified element, but also its descendants. Inheritance allows developers to write less CSS language.

The Box Model is what every programmer should focus on when creating language for CSS. The box model consists of starting from the inside out is the Content, Padding, Border and the Margins. Box sizing, positioning, and adding images are styles to make your webpage your own.

Code, Test, and Refine

A good way to start thinking like a programmer is to use these 5 steps to achieve your webpage.

  • 1. Look for natural boxes
  • 2. Look for repeated styles and syntax
  • 3. Write your HTML
  • 4. Apply you style (from biggest to smallest)
  • 5. Fix things.

Repeat steps 3 to 5 to achieve the webpage you desire.
